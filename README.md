💰 Projeto Conta Bancária - Java (Study Case)
Este repositório contém um projeto de simulação de sistema bancário desenvolvido em Java. O objetivo principal é consolidar fundamentos de Programação Orientada a Objetos (POO) e lógica de programação.

🚀 O que este projeto faz:
Gerencia dados básicos de contas (Titular, Número, Saldo).

Implementa lógica de Depósitos e Saques (com regras de negócio iniciais).

Evolução Atual: Estou refatorando o código para utilizar Arrays (Vetores) para gerenciar múltiplas contas simultaneamente, aplicando conceitos de estruturas de dados.

Projeto inspirado nos desafios práticos do Prof. Nélio Alves.

Em constante evolução: Melhorando a lógica de negócios e a estrutura de dados a cada commit.
